# 项目名称
Ai抠图星睿版
## 项目描述
本项目采用 `models--briaai--RMBG-1.4` 和 `PySide6` 实现图像去背景功能。

## 安装步骤
1. 克隆或下载本项目到本地。
2. 在终端中导航到项目目录：
   ```bash
   cd removebg4
   ```
3. 安装所需的依赖包：
   ```bash
   pip install -r ./model/requirement.txt
   ```
